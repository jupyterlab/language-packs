# Jupyterlab French (France) Language Pack

French (France) language pack for the JupyterLab ecosystem.

## Install

### pip

```bash
pip install jupyterlab-language-pack-fr-FR
```

### conda

```bash
conda install jupyterlab-language-pack-fr-FR -c conda-forge
```
