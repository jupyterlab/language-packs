# Jupyterlab Catalan (Spain) Language Pack

Catalan (Spain) language pack for the JupyterLab ecosystem.

## Install

### pip

```bash
pip install jupyterlab-language-pack-ca-ES
```

### conda

```bash
conda install jupyterlab-language-pack-ca-ES -c conda-forge
```
