# Jupyterlab Spanish (Spain) Language Pack

Spanish (Spain) language pack for the JupyterLab ecosystem.

## Install

### pip

```bash
pip install jupyterlab-language-pack-es-ES
```

### conda

```bash
conda install jupyterlab-language-pack-es-ES -c conda-forge
```
