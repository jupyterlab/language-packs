# Jupyterlab Czech (Czechia) Language Pack

Czech (Czechia) language pack for the JupyterLab ecosystem.

## Install

### pip

```bash
pip install jupyterlab-language-pack-cs-CZ
```

### conda

```bash
conda install -c conda-forge jupyterlab-language-pack-cs-CZ
```
