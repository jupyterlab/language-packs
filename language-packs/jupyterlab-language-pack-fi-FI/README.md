# Jupyterlab Finnish (Finland) Language Pack

Finnish (Finland) language pack for the JupyterLab ecosystem.

## Install

### pip

```bash
pip install jupyterlab-language-pack-fi-FI
```

### conda

```bash
conda install jupyterlab-language-pack-fi-FI -c conda-forge
```
