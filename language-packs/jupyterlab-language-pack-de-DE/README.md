# Jupyterlab German (Germany) Language Pack

German (Germany) language pack for the JupyterLab ecosystem.

## Install

### pip

```bash
pip install jupyterlab-language-pack-de-DE
```

### conda

```bash
conda install -c conda-forge jupyterlab-language-pack-de-DE
```
