# Jupyterlab Greek (Greece) Language Pack

Greek (Greece) language pack for the JupyterLab ecosystem.

## Install

### pip

```bash
pip install jupyterlab-language-pack-el-GR
```

### conda

```bash
conda install -c conda-forge jupyterlab-language-pack-el-GR
```
